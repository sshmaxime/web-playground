# Omniscient

Statistics Explorer for the Omni ecosystem.

https://web-playground-omniscient.vercel.app/

## Get Started

Go to to the root of the Monorepo.

This project uses `Yarn@4.1`, you may need to install it: https://yarnpkg.com/getting-started/install.

Install dependencies:
```
yarn
```

Run `Omniscient`:
```
yarn dev --filter=@web-playground/omniscient
```
